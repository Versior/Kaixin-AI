# 第三方提示词源

系统可从 GitHub 提示词仓库同步图片提示词案例。同步后的内容进入后台提示词管理，再展示到用户侧提示词库。

## 同步内容

通常会解析：

- 标题。
- 提示词正文。
- 预览图。
- 分类。
- 标签。
- GitHub 来源链接。

## 维护约定

- 新增来源前先确认仓库内容格式稳定。
- 同步逻辑应容忍单个仓库失败，避免影响其他来源。
- 后台手工编辑后的提示词不应被无差别覆盖。
- 预览内容可以使用 Markdown，但不要引入不可信脚本。
