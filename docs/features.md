# 功能说明

本文档记录当前系统已经实现的主要功能。项目名称仍沿用 `infinite-canvas`，线上业务侧可称为 Kaixin AI。

## 账号体系

- 支持账号密码注册登录。
- 支持 Linux.do OAuth 登录。
- 支持普通用户和管理员角色。
- 注册时记录用户 `registerIP`。
- 同一 IP 只能注册一个用户。
- 用户有算力点余额，调用云端模型前按模型配置预扣。
- 上游失败、少返回图片或请求被拒绝时会按实际情况退还算力点。

## 生图工作台

`/image` 是用户主要图片创作入口。

支持：

- 文生图。
- 上传参考图后图生图/图片编辑。
- 模型、尺寸、质量和生成张数配置。
- 生成结果预览、下载、保存到素材。
- 本地生成记录回填。
- 云端全站生图任务展示。
- 全站统计排行和用户生图排行。

限流规则：普通用户 3 分钟最多生成 3 张图片，按图片张数计数，不按提交次数计数。管理员不受该限制。

## 全站生图任务

用户端可看到当前全站生图队列状态：

- 正在运行的任务。
- 等待队列。
- 任务用户昵称。
- 模型和生成张数。
- 任务状态。

该区域用于降低用户等待时的不确定感。任务详情不会暴露其他用户的原始提示词、请求体或上游响应。

## 全站统计排行

生图工作台在全站任务下方展示统计排行：

- 总生图。
- 今日生图。
- 成功生图。
- 失败生图。
- 用户生图排行。

排行按成功产出的图片张数排序。限流记录是审计事件，不计入上游生图失败。

## 云端生成历史

`/image/history` 展示当前登录用户自己的云端生成历史。

支持查看：

- 生成时间。
- 模型。
- 类型。
- 状态。
- 生成图片。
- 错误信息。

用户只能看到自己的历史。管理员可在后台查看全站生成日志。

## 灵感画布

`/canvas` 提供自由编排式创作空间。

支持：

- 多个灵感项目。
- 节点拖拽、缩放、复制、粘贴和删除。
- 节点连线和上下游关系。
- 小地图。
- 视图缩放和平移。
- 点阵、网格线、空白背景。
- 浅色和深色主题。
- 撤销和重做。
- 导入导出。

节点类型：图片节点、文本节点、生成配置节点、视频节点。

## 画布 AI 能力

画布里可以从文本、图片或生成配置节点发起 AI 创作。

- 文本节点可以生成或改写文本。
- 文本节点可以一键转成生图配置。
- 图片节点可以直接生成图片，也可以带参考图继续生成。
- 生成配置节点可以汇总上游文本和图片，统一配置模型、尺寸、质量和张数。
- 视频节点可以承载上传视频或生成视频结果。

如果参考图丢失或文件为空，系统会在请求前拦截，避免发出无效图生图请求。

## 视频创作台

`/video` 提供独立视频创作页面。

支持：

- 输入提示词生成视频。
- 上传最多 7 张参考图。
- 设置清晰度、尺寸和秒数。
- 展示生成结果。
- 保存到素材。
- 下载视频。
- 本地历史回填。

视频接口参数会在前端归一化，避免残留图片参数影响视频请求。

## 素材库

系统包含用户侧“我的素材”和后台素材库。

用户侧素材用于沉淀个人图片、视频和创作结果。后台素材库用于维护公共素材资源。

媒体文件通过本地存储保存，业务 JSON 只保存 `storageKey` 和元信息，不长期保存大体积 base64。

## 提示词库

提示词库会从多个 GitHub 仓库同步图片提示词案例。

后台可管理提示词：

- 新增、编辑、删除。
- 多选批量删除。
- 分类和标签。
- 封面和 Markdown 预览。
- 从配置的提示词源同步。

## 管理后台

`/admin` 是管理员入口。

主要模块：

- 用户管理。
- 算力点日志。
- 提示词管理。
- 素材管理。
- 生成日志。
- 系统设置。

系统设置包含公开配置和私有配置。公开配置会返回给前端，私有配置只给管理员和后端使用。

## 模型渠道

后端支持多个 OpenAI 兼容模型渠道。

管理员可以配置：

- 渠道名称。
- Base URL。
- API Key。
- 可用模型。
- 启用状态。

用户请求云端模型时，后端按模型名匹配可用渠道并转发请求。

## 公告弹窗

公开配置支持网站公告弹窗：

- `enabled`：是否启用。
- `title`：标题。
- `content`：内容。
- `version`：公告版本。
- `oncePerVersion`：同版本是否只弹一次。

前端按版本记录用户已读状态。
