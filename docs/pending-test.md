# 待测试清单

本文档记录已经实现但仍值得人工回归的功能点。自动测试通过不代表这些交互细节都已覆盖。

## 账号和权限

- 普通注册成功后记录 `registerIP`。
- 同一 IP 再次注册时被拦截。
- Linux.do 首次登录注册同样受 IP 限制。
- 未登录访问用户接口时正常提示未登录。
- 普通用户无法访问管理员接口。

## 生图工作台

- 一次生成 1 张，连续 3 次后第 4 张触发限流。
- 一次生成 3 张后，3 分钟内再次提交触发限流。
- 管理员不受 3 分钟 3 张限制。
- 文生图成功后扣除对应算力点。
- 图生图上传空图片时直接提示图片文件为空。
- 上游少返回图片时只扣实际成功部分，其余退点。
- 本地历史能回填提示词、参考图、模型、质量、尺寸和张数。
- 云端历史只展示当前账号记录。

## 全站任务和统计

- 用户端全站生图任务区域展示运行中和排队任务。
- 统计排行位于全站生图任务下方。
- 用户生图排行按成功图片张数排序。
- 限流记录不会被误当成上游生图失败。

## 画布

- 灵感项目导出 ZIP 包含 `projects.json` 和引用媒体。
- 导入 ZIP 后图片、视频和节点关系恢复正常。
- 删除图片节点后撤销，图片内容仍能恢复。
- 生成配置节点能读取上游文本和图片。
- 图片尺寸、质量和数量能正确传给图片接口。
- 参考图丢失时重试会直接提示，不发送空请求。

## 视频

- `/video` 能创建视频任务。
- 清晰度、尺寸和秒数会被归一化为视频接口支持的参数。
- 参考图最多 7 张。
- 创建失败时展示后端 `msg`，不继续轮询空任务 ID。
- 生成记录能回填提示词、参考图和参数。

## 后台

- 管理员能调整用户算力点并生成流水。
- 管理员能查看生成日志。
- 管理员能配置模型渠道和模型价格。
- 系统公告弹窗能按版本只弹一次。
- 提示词源同步失败时不影响已有提示词。
