# 画布节点手册

灵感画布由节点和连线组成。节点承载文本、图片、视频或生成配置，连线表达上下游引用关系。

## 文本节点

文本节点用于保存提示词、草稿、说明文案和 AI 生成的文字结果。

常用操作：

- 双击文本内容直接编辑。
- 选中后点击工具栏“编辑文字”。
- 使用字号按钮调整文本大小。
- 打开节点下方对话框生成或改写文本。

当文本节点为空时，下方对话框会把生成结果回填到当前节点。已有内容时，改写结果会生成到右侧新文本节点，并自动连线。

## 图片节点

图片节点用于展示上传图片、素材图片、生成图片和编辑结果。

常用操作：

- 上传或拖入图片。
- 替换图片。
- 下载图片。
- 保存到我的素材。
- 裁剪图片。
- 本地变换图片。
- 继续用当前图片作为参考图生成新图。

生成结果数量为 1 时，结果可直接回填当前节点。数量大于 1 时，会生成图片组或多个结果节点。

每个生成结果会记录提示词、生成类型、模型、尺寸、质量、数量和参考图引用，方便后续重试。

如果参考图丢失、无法读取或为空，系统会直接提示，避免发送无效请求。

## 生成配置节点

生成配置节点用于汇总上游文本和图片，统一配置 AI 生成参数。

它可以读取：

- 上游文本节点内容。
- 上游图片节点内容。
- 自己的提示词和参数配置。

可配置项：模型、尺寸、质量、生成张数。适合把多个灵感节点组合成一次完整图片请求。

## 视频节点

视频节点用于展示上传视频或 AI 生成视频。

支持：

- 上传本地视频。
- 原生播放器播放、暂停和拖动进度。
- 从空视频节点输入提示词生成视频。
- 从文本、图片或配置节点创建视频生成节点。

视频生成接口使用 OpenAI 兼容风格的视频接口，并在前端归一化清晰度、尺寸和秒数。

## 推荐流程

1. 新建文本节点，写下创作想法。
2. 用文本节点下方对话框优化提示词。
3. 把优化结果连接到生成配置节点。
4. 添加参考图片节点。
5. 在生成配置节点里设置模型、尺寸、质量和张数。
6. 生成图片，挑选结果保存到素材。
