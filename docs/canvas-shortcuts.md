# 画布快捷键

本文档记录灵感画布常用鼠标和键盘操作。

## 视图

- 拖动画布空白处：平移视图。
- 鼠标滚轮：缩放画布。
- 缩放滑杆：精确调整缩放。
- 重置视图：回到默认缩放和居中位置。

## 选择

- `Ctrl / Cmd` + 拖动：框选多个节点。
- `Shift / Ctrl / Cmd` + 点击节点：追加或取消选择。
- `Ctrl / Cmd` + `A`：全选节点。
- `Esc`：取消选择，并关闭当前浮层。

## 编辑

- `Ctrl / Cmd` + `C`：复制选中节点。
- `Ctrl / Cmd` + `V`：粘贴节点。
- `Delete / Backspace`：删除选中节点或连线。
- `Ctrl / Cmd` + `Z`：撤销。
- `Ctrl / Cmd` + `Shift` + `Z`：重做。
- `Ctrl / Cmd` + `Y`：重做。

## 图片和素材

- 拖入图片文件：上传图片到画布。
- 导入图片按钮：从本地选择图片。
- 素材库或我的素材：选择素材后插入画布。
- 图片节点工具栏：可下载、保存素材、裁剪、变换或继续生成。

## 撤销和重做范围

撤销和重做会记录节点、连线、视口、背景模式和助手会话变化。

不纳入当前画布历史的内容：

- 项目名称。
- 登录状态。
- 全局 AI 配置。
- 我的素材保存操作。
- 后台配置。
