# CHANGELOG

## Unreleased

## v0.1.0-kaixin.29 - 2026-05-29

- 用户管理页将 Linux.do 标识列改为注册 IP，并支持按注册 IP 搜索。
- 顶部版本号改为短显示，版本弹窗保留完整版本号。
- 版本检查改为读取 Kaixin-AI 仓库的真实版本和更新内容。
- 生图限流文案统一为 3 分钟最多 3 张，并覆盖画布生图入口。

## v0.1.0-kaixin.28 - 2026-05-28

- 修复用户侧页面右上角算力点余额展示。
- 新增后台公告弹窗配置，支持按版本控制是否重复展示。

## v0.1.0-kaixin.27 - 2026-05-28

- 优化生成日志、素材库和提示词后台维护体验。
- 完善线上 Docker 部署配置与运行稳定性。

## v0.1.0-kaixin.26 - 2026-05-28

- 修复生图请求 normalize 时覆盖 `n` 的问题，保留用户原始生成张数。
- 生图限流改为按图片张数计数：3 分钟最多 3 张。
- 图生图上传空图片时在本系统直接拦截，不再发送给上游。

## v0.1.0-kaixin.25 - 2026-05-28

- 用户端生图工作台展示全站生图任务和全站统计排行。
- 注册时记录 `registerIP`，同一 IP 只能注册一个用户。
- Linux.do 登录注册链路同步应用注册 IP 限制。

## v0.1.0-kaixin.24 - 2026-05-28

- 新增全站生图统计接口。
- 统计总生图、今日生图、成功生图、失败生图和用户生图排行。

## v0.1.0-kaixin.23 - 2026-05-28

- 新增用户端云端生图历史页面 `/image/history`。
- 新增账号级历史接口 `/api/v1/images/history`，只返回当前登录用户的生成记录。

## v0.1.0 - 2026-05-26

- 优化画布和素材导出导入。
